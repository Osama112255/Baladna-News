أرشيف موقع إخباري — نسخة متكاملة (محلي)
----------------------------------------
المحتويات:
- index.html        -> الصفحة الرئيسية
- style.css         -> ملف التصميم الحديث (ألوان علم مصر، أنيميشن)
- script.js         -> جافاسكربت: يجلب أخبار Google News RSS عبر api.allorigins.win، يدعم الأقسام والبحث وModal
- articles.json     -> ملف تجريبي (100 خبر) كنسخة احتياطية / fallback
- assets/           -> صور مكانية (logo, placeholders, flag)

تشغيل محلي (مهم - لتجاوز قيود CORS):
1) افتح موجه الأوامر (CMD) داخل المجلد:
   cd Desktop\akhbar_site_full    (أو مكان المجلد عندك)
2) شغّل سيرفر بسيط:
   python -m http.server 8000
3) افتح المتصفح وزر:
   http://localhost:8000

شرح سريع:
- الموقع يستخدم Google News RSS (بحث) لجلب أحدث الأخبار باللغة العربية. يتم استدعاء الرابط عبر خدمة api.allorigins.win لتجاوز قيود CORS عند فتح الموقع محليًا.
- عند الضغط على أي قسم (أو بطاقة مصادر سريعة) يتم جلب وفلترة الأخبار المناسبة لتلك الفئة.
- عند الضغط على أي خبر يظهر Modal جميل داخل الموقع بدلاً من فتح الموقع الخارجي.
- إذا فشل جلب RSS لأي سبب، الموقع لا يزال يعمل باستخدام articles.json كنسخة احتياطية.

ملاحظة عن حقوق الصور والمصادر:
- الموقع يعرض عناوين/ملخصات وروابط للأخبار الأصلية (مصادر خارجية). إذا رغبت بنشر الموقع على الإنترنت علناً، تأكد من سياسات إعادة الاستخدام لكل مصدر.
